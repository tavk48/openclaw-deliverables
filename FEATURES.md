### New Task Button Feature

**Overview**: This feature introduces a 'New Task' button that enables users to create tasks quickly without navigating through multiple screens.

**Details**:
- **Button Functionality**: When clicked, the button opens a prompt/modal allowing users to enter task details.

- **Fields Included**:
   - **Title**: A short text input for the task title.
   - **Description**: A larger text area for a detailed description of the task.
   - **Due Date**: A date picker for selecting the deadline for the task.
   - **Priority Level**: A dropdown to choose the priority (e.g., low, medium, high).

**Usage**: Users can easily create a new task with this feature, streamlining the task management process.
